TODOバトル-AI搭載のゲーム感覚タスク管理アプリ

単なるタスク管理ではなく、楽しく続けられることを重視したアプリです。
タスクを入力するとAIが所要時間を推定し、ユーザーはタイマーに挑戦します。
時間内に完了ボタンを押せば「クリア！」となり、達成感を得ながら作業を進められるアプリです。
特徴
  • AIがタスク時間を自動推定
  • 熱血コーチが応援（表情・セリフが時間経過で変化）
  • 楽しくタスクをこなせるゲーム感覚の設計
制作背景
「他人に時間を決められて作業するとパフォーマンスが上がる」という体験をもとに開発しました。
使用技術
フロント：React, HTML, CSS
バックエンド：Node.js
AI：OpenAI API
デプロイ：Vercel
アプリはこちらからご利用いただけます：
https://1cool-app-9bbv.vercel.app/
